You can email questions about ACL2 usage to the acl2-help mailing list: acl2-help@lists.cc.utexas.edu. If you have more general questions about ACL2, for example, about projects completed using ACL2, you may prefer the acl2 mailing list, acl2@lists.cc.utexas.edu, which tends to have wider distribution.
