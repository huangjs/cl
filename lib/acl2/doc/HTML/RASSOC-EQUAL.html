RASSOC-EQUAL

look up value in association list, using equal as test 
Major Section:  PROGRAMMING

(Rassoc-equal x alist) is similar to (assoc-equal x alist), the difference being that it looks for the first pair in the given alist whose cdr, rather than car, is equal to x. See assoc-equal.

The guard of rassoc-equal requires its second argument to be an alist. See rassoc and see rassoc-eq.
