REMOVE-EQUAL

remove all occurrences, testing using equal 
Major Section:  PROGRAMMING

(remove-equal x l) is l if x is not a member of l as tested with member-equal, else is the result of removing all occurrences of x from l.

The guard for (remove-equal x l) requires l to be a true list.

Also see remove1-equal, see remove, and see remove-eq.
