REMOVE1-EQUAL

remove first occurrences, testing using equal 
Major Section:  PROGRAMMING

(remove1-equal x l) is l if x is not a member of l as tested with member-equal, else is the result of removing all occurrences of x from l.

The guard for (remove1-equal x l) requires l to be a true list.

Also see remove-equal, see remove1, and see remove1-eq.
