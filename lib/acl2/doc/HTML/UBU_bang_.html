UBU!

undo commands, without a query or an error 
Major Section:  HISTORY


Example:
:ubu! :x-4

The keyword command :ubu! is the same as :ubu, but with related queries suppressed appropriately, and with a guarantee that it is ``error-free.'' More precisely, the error triple returned by :ubu! will always have a first component of nil. :Oops will undo the last :ubu!. Also see ubu, see ubt, and see u.
