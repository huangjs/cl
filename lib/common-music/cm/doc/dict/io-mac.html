[Macro]
(io stream {keyword value}*)

Initializes stream according to its keyword arguments. Stream can be an event stream or a string that names the stream.

Examples:

Example 1. Initializing a MIDI file. 

;;; setting the tempo and time signature of a MIDI file.
(io "test.mid" :tempo 120
    :time-signature '(3 4))

Example 2. Initializing a CLM audio file. 

;;; setting the srate, channel and reverb of an AIFF file

(io "/tmp/piece.aiff" :srate 44100 :channels 2
    :reverb 'nrev)
