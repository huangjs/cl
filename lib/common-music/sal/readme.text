1. Install SAL sources somewhere, call it "/path/to/sal/"

2. Add (load "/path/to/sal/sal.el") to your .emacs file.

3. Start Emacs with Slime and then start CM under slime.
   ('M-x cm' if you use cm/etc/xemacs/cm.el)

4. Load sal into CM: (use-system :sal :directory "/path/to/sal/")

5. Visit examples: C-x C-f /path/to/sal/examples.sal 
   The resuting buffer should be in SAL mode with SAL syntax
   highlighting active

6. Work through the examples using directions at top of file.
