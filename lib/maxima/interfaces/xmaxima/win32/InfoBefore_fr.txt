1. Les utilisateurs Windows 9x devraient lire la
    section espace d'environnement du fichier
    README.

2. Si l'interface utilisateur (GUI) ne fonctionne pas
    lisez la section du fichier README consacré
    au pare-feux.

3. Si non seulement l'interface graphique de Maxima mais 
   même Maxima en ligne de commandes ne fonctionne pas, 
   lire le paragraphe du README sur la Prévention de 
   l'Exécution des Données (PED).
