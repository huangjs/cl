C Software

There are two recommended ways to compile and run C code in this course, if you are on a Windows machine.

  • Download and install the Dev-C++ compiler. This is similar to Visual Studio (see below), but smaller and completely free. It has some limitations, but is more than enough for our needs.
  • Use Microsoft Visual Studio for C++ 6.0, which is already installed in most of the labs on campus.

How to submit your work

  • Only submit the .C and (if any) .h files that you have written.
  • Combine the files to be submitted into a zip file named firstname_lastname_pa?.zip where ? is the number of the current assignment. To zip the files use a utility like Winzip Filzip, or TUGzip. If you use the built-in file compressor in Windows, make sure you know what you're doing!
  • Email your file as an attachment to c-riesbeck@northwestern.edu. Make sure you copy the email to yourself and then check to see whether the attachment has been sent correctly.
  • You should receive a confirmation within 24 hours at the latest. This will only acknowledge receipt of your email. We will not check the attachment at that time, so make sure everything is there. If there is any problem, email me at c-riesbeck@northwestern.edu. Important: Put "EECS 110:" in the Subject line.

How to get help

Programming can be very time consuming. Always start your assignments as early as possible. If you do get stuck, do NOT waste hours on a single error. Instead, get help. The fastest way is to check the newsgroup. Chances are someone else has had the same problem as you and can give you a hint. Tips, clarifications and answers will also be posted by the staff. You may discuss the homework with other students in the newsgroup as long as you never post anything that will be graded.

The TAs will have regular office hours at the lab, where you can get them to look at your work in progress and offer you hints and useful programming tips. Take advantage of that.
