Microsoft Visual Studio is a powerful integrated development environment (IDE) for C, C++, and many other languages on the Windows platform. The teaching lab has Microsoft Visual Studio installed. You can also get a temporary copy from the MSDNAA site, if you have a valid EECS account.

System requirements

Visual Studio requires Windows, preferrably XP or later, and several hundred megabytes of disk space.

Installing Visual Studio

Before doing anything else, in Windows, create a folder (directory) where you will keep all your EECS 110 code, e.g., C:\CS110\Code. Then get and install Visual Studio, following the instructions at the MSDNAA site.

Creating a Project

In Visual Studio and most other IDE's, you create a project for each application you want to build. A project is a small file that contains a list of the code files needed to make the application, plus various settings for compiling those files. The steps below show how to create a project in Visual Studio. You do basically the same thing in other IDE's, though the menu names may be slightly different.

The simplest kind of project to build in Windows is one that does all its input and output in a console window, i.e., the Command Prompt window in Windows (what used to be called a DOS window in Win98). Such programs can be simpler to set up, but are a bit old-fashioned.

  • Start Visual Studio.
  • On the main screen, click on the File menu, then on New, then on Project. The New Project dialog box will appear.
  • On the dialog box, click on Visual C++ Projects. The C++ Project Templates dialog box will appear.
  • On the dialog box, click on Win32 Console Project.
  • In the Name field, enter a name for your project, for example PA1 for Programming Assignment #1.
  • Change the directory for the project to your EECS 110 code directory.
  • Click OK. The Project Settings dialog box will appear.
  • Click on Application Settings, then on Empty Project, then on Finish. This creates an empty project and opens up the main project interface. Now you need to add some code to it.
  • Click on the Project menu, then on Add new item, then select C++ File.
  • Enter the name for your file, e.g., pa1_1.c for Programming Assignment #1, Problem 1. Important: be sure to end the file name with .c. That tells Visual Studio you really want a C file, not a C++ file.
  • Click OK. An editing window will appear for the file you just created. Just click in that window and start entering your code.
  • Once your code is entered, you can compile it by clicking on the Build menu and then on Build Project. Visual Studio will save and compile all the code files in the project.
  • If your code compiles and links without error, you can run it by clicking on the Build menu and then on Start without debugging. Visual Studio will run your program.
