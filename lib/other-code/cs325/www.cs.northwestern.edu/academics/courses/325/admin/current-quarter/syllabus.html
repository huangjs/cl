Syllabus

Because this course is primarily an individually-driven series of programming experiences and feedback, it doesn't have a syllabus in the normal sense of a sequence of topics rigidly followed for all students till death or final exam do you part.

There are however some key topics in (AI) programming that I do want everyone to encounter, and you will encounter them this year in roughly this order:

  • test-first programming and test-driven development, using the CS 325 lisp-unit package.
  • Lisp-based web servers and web clients, using (portable) AllegroServe
  • frame-based knowledge representation techniques
  • XML-based knowledge sharing and the Semantic web
