Buttons

Making radio buttons

  1. Make some two-pict buttons.
  2. Double click on each of them in turn and make the following changes:
    1. Change the "group" string on each to something mnemonic. Use the exact same string each time.
    2. Check each of them as "sticky". Now when you click on one, it will stay down, but the other ones will pop up.
